# Claude Code Academy

> *From first run to production-grade workflows — a practitioner's field guide.*

Ten focused lessons covering the full Claude Code workflow: navigating codebases, writing effective prompts, debugging, testing, refactoring, and context management. Each lesson is a self-contained reference page you will return to after the first read.

**Format:** 10 lessons · 4 phases · 5–8 days  
**Best for:** Engineers and tech leads using (or about to use) Claude Code on real projects
