# Agentic AI Academy

> *Technology-agnostic, concept-first — the ideas here remain true as tools change.*

A 16-lesson progression through the building blocks of modern AI: how LLMs reason, what agents actually are, how multi-agent systems are designed, and how to ship them reliably in production. Built on fundamentals, not frameworks — so every future tool you pick up will feel familiar.

**Format:** 16 lessons · 5 sections  
**Best for:** Engineers and product leads who want the *why* before the *how*
